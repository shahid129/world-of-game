About Game

'Guess the number' is a game that I have created, where a number is generated randomly between 1 and 20. The user is given five chances to guess the correct number within thirty seconds. Each time the users clicks a number, a hint is provided. This hint tells the user to either choose a higher, or a lower number. This game has elements of both reasoning and luck.

This game is intended for people of any age and I hope that everyone enjoys it. Happy guessing everyone.
